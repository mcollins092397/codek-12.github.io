Lesson 2, Part 2

User input

Declaring variables is a useful skill to know but the true power of variables comes with their ability to hold input from the user, do calculations, operations, and print information that can change

Lets start by figuring out how we can take in input from the user and store it in variables.

In order to take in input we first need to import the scanner class.

The first data type that we will cover is "int"
"int" is short for integer and is used to represent whole numbers (...-3,-2,-1,0,1,2,3...). int has a default value of 0 so if not assigned that is what it will hold.
In the compiler to the right we can see how an int is declared as well as how it can be printed

The next data type that we will cover is "double"
"double" functions very similarly to int. It is used to represent numbers with decimal, but can still be assigned whole numbers and negative numbers. double has a default value of 0.0

The next data type that we will cover is "boolean"
"boolean" is used to represent true and false. These are the only two values that a boolean can hold. The default value of a boolean is false

The next data type is "char"
char stands for character and is used to represent a single character. This character can be a letter, number, or symbol. The default value for char is '\u0000' which represents a null character

The final data type that we will cover is "String"
"String" is used to represent a group of characters. It is commonly used to store words and sentences. The default value for a String is null
