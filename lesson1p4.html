Now you may be asking yourself, what does System.out.println actually mean?
System is a built in Java class that contain useful tools that our program can use
out is short for "output"
println is short for "print line" and tells the statement to print what is given in the brackets and then move to the next line of the output
Then we put what we want printed in the brackets ( ) and since we are printing a sentence we put the words in parentheses " "
Then finally we put a semicolon ; at the end of the line

NOTE! While programming in Java make sure that you remember to put semicolons at the end of your statements or your program will fail to run!
