Lesson 1, Part 2

Basic .java file layout

Now that we have discussed what Java is, lets begin our dive into how to actually create a program in Java!

We will start with a simple program that says "Hello World!" to the user.

To do this we need to first create the class of the java file.
Think of a class as a box. This box holds all ideas relating to a certain topic or goal. For example, with the class for this project we will include everything needed to print "Hello World!" to the user. But the topic of the box can vary to just about anything, from describing a car, to the information of a bank account, and much much more.

A basic class (and the class for this project) looks something like this:

As you can see the syntax for declaring a class is as follows:
public class ClassName

Don't worry about what the "public" stands for right now we will cover that in later lessons. But the word "class" lets java know that we are declaring a class. Then we simply put what we want the name of the class to be after. With our example the name of the class is "HelloWorld"

With classes we want the name of the .java file itself to match the name of the class in it, so keep that in mind

You may have also noticed the pair of curly braces {}. We will refer to the area between these two braces as the body of the class.

A class's body contains all of the code required for that class to operate. So for our example we would put all of the code that prints out "Hello World" inside of the curly braces.

Now inside of our "HelloWorld" class we will put a method named "main". A method is a block of code that runs whenever its name is called in the program.

The "main" method is a special method where we put all of the code that we want to run when we start the program. Notice that it is inside of the "HelloWorld" class. With this method, everything inside its body (shown by its curly braces {}) will run when we run the "HelloWorld" .java file.

Also notice that the "main" method has a body of its own and its own slightly different declaration statement. With this method we have the same "public" word as our class as well as a "static" and "void". All of which we will discuss later on in the course. Then we declare the method name "main" and put (String[] args) in parentheses after it. "String[] args" is a placeholder for information that we want to pass to the program and is necessary for the "main" method to run. But we will discuss what it actually means later on in the course. For now just remember to include it!
