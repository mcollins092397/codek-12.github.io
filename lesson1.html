Lesson 1, Part 1

What is Java?

Welcome to Code K-12! In this course we will learn the basics of coding in Java. But before we begin lets talk about what Java actually is.

- Java is a programming language that lets us write code for computers to run.

- Java is a general purpose language that was created by James Gosling in 1995 but remains one of the most popular programming languages today.

- Java is a high-level, class-based, and object-oriented programming language but lets not worry about what this means yet. We will discuss what this means later on in the course.

- Java programs are made up of statements called syntax that describe specific actions that we want the computer to complete. These syntax statements are all contained within a program, which is a file that our computer runs in order to preform these actions
