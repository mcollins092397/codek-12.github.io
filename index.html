Code K-12

Learn to code with Java

Start Now!

Code K-12 is a programming course designed to teach K-12 students (with an emphasis on middle and high school students) the basics of coding in Java!

Throughout this course students will learn everything needed to create basic programs. This is taught through a series of lessons that are scattered with examples, videos, questions, and even the occasional project.
